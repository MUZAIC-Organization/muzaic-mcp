# Changelog

## [1.0.0] — 2025-03-10

### Added
- Initial release of Muzaic MCP Server
- `muzaic_get_tags` — list available music tags
- `muzaic_generate_music` — generate single track with parameters & keyframes
- `muzaic_create_soundtrack` — multi-region soundtracks
- `muzaic_regenerate` — regenerate from hash
- `muzaic_validate_tags` — check tag compatibility
- `muzaic_account_info` — check token balance
- Support for Claude Desktop, Cursor, VS Code, Claude Code
- PyPI package: `pip install muzaic-mcp` / `uvx muzaic-mcp`
